Pregunta 1: Optimización con Random Forest y Pipelines Una empresa de logística internacional busca predecir el tiempo de entrega de sus contenedores. El dataset presenta retos específicos: las variables tienen escalas muy diferentes (distancia en km, peso en toneladas, número de aduanas), lo que requiere un preprocesamiento riguroso. El equipo de arquitectura de datos exige el uso de un Pipeline para garantizar que el escalado de datos se realice correctamente dentro de cada fold de la validación cruzada, evitando así el data leakage (fuga de información). Tu Misión Escribe una función llamada optimizar_modelo_logistica(X, y) que implemente un flujo de trabajo automatizado siguiendo estas especificaciones: 
1. Construcción del Pipeline: Crea un objeto Pipeline de sklearn que combine: o Un paso de escalado llamado "scaler" usando StandardScaler. o Un paso de regresión llamado "rf" usando RandomForestRegressor(random_state=42). 
2. Malla de Parámetros (Grid): Define un diccionario para GridSearchCV que optimice los siguientes hiperparámetros del bosque (recuerda usar el prefijo del paso rf__): o n_estimators: [50, 100] o max_depth: [None, 5, 10] 
3. Búsqueda Sistemática: Configura un GridSearchCV con: o El pipeline anteriormente creado. o Validación cruzada de 3 folds. o Métrica de evaluación: MAE (scoring='neg_mean_absolute_error'). 
4. Entrenamiento: Ajusta el modelo completo utilizando los datos X e y. 
Requisitos Técnicos • Librerías: pandas, numpy y sklearn. • Componentes: Pipeline, StandardScaler, RandomForestRegressor y GridSearchCV. 
Retorno La función debe devolver un diccionario con la siguiente estructura: Python { "mejor_modelo": pipeline_entrenado, # El mejor estimador (Pipeline completo) "mejor_mae": float # El valor absoluto del mejor MAE obtenido }
